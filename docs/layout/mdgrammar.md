# Markdown 
Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。由于其简洁、通用的特性，Markdown 已成为程序员、博客作者和文档撰写者的首选。

以下是 Markdown 的完整语法手册，分为 **基础语法**、**扩展语法** 和 **常用技巧**。

---

### 基础语法

#### 标题
使用 `#` 号标记，`#` 的数量代表标题级别（1-6 级）。
```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

#### 字体格式
*   **加粗**：使用两个 `*` 或 `_`
*   **斜体**：使用一个 `*` 或 `_`
*   **加粗并斜体**：使用三个 `*` 或 `_`
*   **删除线**：使用两个 `~`
```markdown
**这是加粗文字**
*这是斜体文字*
***这是加粗斜体***
~~这是删除线~~
```

#### 列表
*   **无序列表**：使用 `-`、`+` 或 `*` 后面加空格。
*   **有序列表**：使用数字加点 `1.` 后面加空格。
```markdown
- 无序项目 A
- 无序项目 B

1. 第一步
2. 第二步
```

#### 链接与图片
*   **链接**：`[显示文本](URL "可选标题")`
[百度](https://www.baidu.com)
*   **图片**：`![替代文字](图片地址 "可选标题")`
![Markdown图标](https://markdown-here.com/img/icon256.png)

```markdown
[百度](https://www.baidu.com)
![Markdown图标](https://markdown-here.com/img/icon256.png)
```

#### 引用
使用 `>` 号。
> 这是一个引用块。
>> 这是嵌套引用。
```markdown
> 这是一个引用块。
>> 这是嵌套引用。
```

#### 代码
*   **行内代码**：使用反引号 `` ` ``。
*   **代码块**：使用三个反引号 ` ``` `，并可指定语言进行语法高亮。
````markdown
使用 `printf()` 函数。

```python
def hello():
    print("Hello World")
```
````

#### 分割线
使用三个或更多的 `*`、`-` 或 `_`。
```markdown
---
***
___
```

### 扩展语法 (GFM 等常用版本支持)

#### 表格
使用 `|` 分隔单元格，使用 `-` 分隔表头。
*   `:---` 左对齐（默认）
*   `---:` 右对齐
*   `:---:` 居中对齐
```markdown
| 姓名 | 年龄 | 职业 |
| :--- | :--: | ---: |
| 张三 | 25 | 程序员 |
| 李四 | 30 | 设计师 |
```
| 姓名 | 年龄 | 职业 |
| :--- | :--: | ---: |
| 张三 | 25 | 程序员 |
| 李四 | 30 | 设计师 |

#### 任务列表
使用 `- [ ]` 表示未完成，`- [x]` 表示已完成。
```markdown
- [x] 完成文档编写
- [ ] 检查代码逻辑
```
- [x] 完成文档编写
- [ ] 检查代码逻辑

#### 数学公式
大多数编辑器（如 Typora, Obsidian）支持 LaTeX 语法。
*   **行内公式**：使用 `$E=mc^2$` $E=mc^2$
*   **块级公式**：
```markdown
$$
\sum_{i=1}^{n} a_i = 0
$$
```

#### 注脚 
用于对文本进行补充说明。
```markdown
这是一句需要解释的话[^1]。

[^1]: 这是注脚的具体内容。
```
这是一句需要解释的话[^1]。

[^1]: 这是注脚的具体内容。

#### 目录
在支持的编辑器中输入 `[TOC]` 会自动根据标题生成目录。

### 常用转义与排版技巧

#### 表情和特殊符号
**1. HTML 实体引用**

|符号 |HTML代码|符号名称|
| :---: | :---: | :---: |
|©	|`&copy;`	|版权符号|
|®	|`&reg;`	|注册商标|
|™	|`&trade;`	|商标|
|×	|`&times;`	|乘号|
|÷	|`&divide;`	|除号|
|±	|`&plusmn;`	|正负号|
|§	|`&sect;`	|章节符号|
|←	|`&larr;`	|左箭头|
|↑	|`&uarr;`	|上箭头|
|↓	|`&darr;`	|下箭头|
|→	|`&rarr;`	|右箭头|
|(空格)	|`&nbsp;`	|不换行空格|

**2. emoji 短代码**
如 `:smile:` 会显示 😄，`:heart:` 会显示 ❤️。
也可以直接通过输入法（Windows：Win + ; 或 ctrl + .）或复制粘贴来插入 emoji。
更多 emoji 短代码可以参考 [Emoji Cheat Sheet](https://www.webfx.com/tools/emoji-cheat-sheet/)。

**3. latex符号**
详见 [LateX数学公式](https://stquasimodo.github.io/notebook/layout/latex_math.html) 页面。

#### 转义字符
如果你想在文档中直接显示 Markdown 语法符号（如 `*`），需要使用反斜杠 `\`。
```markdown
\*这不是斜体\*
```

#### 换行
*   **普通换行**：在行末输入**两个空格**然后回车。
*   **强制换行**：直接使用 HTML 标签 `<br>`。

#### 字体颜色与大小 (HTML)
Markdown 原生不支持改颜色，但可以嵌入 HTML。
<font color="red">这段文字是红色的</font>
<center>这段文字居中</center>

```markdown
<font color="red">这段文字是红色的</font>
<center>这段文字居中</center>
```

#### 自动链接
使用 `< >` 包裹 URL。
<https://www.google.com>

```markdown
<https://www.google.com>
```
